
## App Name: Query-Based Consolidation

This folder contains Business Central projects related to Usages of Query objects for consolidation of records in Business Central.


## Short Description: 

Consolidate Business Central data using Query Objects with grouping and aggregation.

## Typical Use Cases

- Writing Query Objects.
- Using Query Objects to Consolidate records in Business Central Data. 
- Using Query helps not to Loop Records via Code.

## Intended Audience

- Business Central Developers  
- Technical Consultants  
- Solution Architects  
- Learners exploring real-world implementations
