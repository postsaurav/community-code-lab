
## App Name: Item Template Attributes

This folder contains Business Central projects related to Enables defining Item Attributes at template level in Business Central.


## Short Description: 

Enables defining Item Attributes at template level.

## Typical Use Cases

- Extends Item Templates
- Enables defining Item Attributes at template level
- Attributes are applied when items are created from templates

## Intended Audience

- Business Central Developers  
- Technical Consultants  
- Solution Architects  
- Learners exploring real-world implementations
